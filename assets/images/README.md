# Assets Directory

## Images

Place your image files in this directory.

### Required Files:

**`logo.png`** - Your BoyaQueue logo/icon
- Recommended size: 96x96px to 256x256px (will be displayed at 48x48px on mobile, 56x56px on desktop)
- Format: PNG with transparency preferred
- Should work well on the clay terracotta gradient background (#C2794D to #8B5A3C)
- The image will have a drop shadow and a fun 360Â° rotation animation on hover

### Tips for best results:
- Use a square aspect ratio (1:1)
- Include some padding/margin in your image if it's a tight crop
- Test on both light and dark backgrounds
- PNG format with transparency gives the best results
